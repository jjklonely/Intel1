# Учебный сайт о компании Intel

Образовательный веб-сайт с информацией о компании Intel и процессорах Intel.

## Функциональность

- ✅ Главная страница с информацией о компании Intel
- ✅ Интерактивный слайдер поколений процессоров Intel (15 поколений)
- ✅ Переключение языков: Русский (RU), Кыргызский (KG), Английский (ENG)
- ✅ Круглая кнопка с иконкой ИИ для задавания вопросов
- ✅ Интеграция с ChatGPT API (GPT-3.5) для ответов на вопросы
- ✅ Современный и чистый дизайн

## Установка и настройка

1. Откройте файл `script.js`
2. Найдите строку: `const API_KEY = 'YOUR_API_KEY';`
3. Замените `YOUR_API_KEY` на ваш реальный API ключ OpenAI
4. Сохраните файл

### Как получить API ключ OpenAI:

1. Перейдите на https://platform.openai.com/
2. Зарегистрируйтесь или войдите в аккаунт
3. Перейдите в раздел API Keys
4. Создайте новый API ключ
5. Скопируйте ключ и вставьте в файл `script.js`

## Использование

1. Откройте файл `index.html` в браузере
2. Используйте кнопки переключения языков в верхней части страницы
3. Просматривайте информацию о процессорах через слайдер
4. Нажмите на круглую кнопку с иконкой ИИ в правом нижнем углу для задавания вопросов
5. Введите вопрос и получите ответ от ChatGPT

## Структура файлов

```

├── styles.css      # Стили и дизайн
├── script.js       # JavaScript функциональность

```

## Примечания

- Для работы функции задавания вопросов необходим активный API ключ OpenAI
